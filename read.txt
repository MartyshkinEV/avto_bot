Описание проекта создание бота для автовладельцев
Мешает авто – напиши владельцу.
Для начала распространим бота в двух общих чатах дома 40 и 40.1


Фукционал бота,
1. Регистрация автовладельца – заходит в бота и регистрирует его на свое авто просто вводит номер авто
2. Получает сообщение для автовладельца и отправляет его


Напрямую не контактируют просто пересылают сообщения через бота

Работа бота будет происходить а сервере
Время написания бота 3 дня

Базой для бота будет монго дб

для написания бота используем Pycharm и сервер на bitwebe

назовем SMFC – send message for car
